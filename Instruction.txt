Run CityLink Frontend on Another PC (Vite + React)


cd CityLink/client


1) Install required software

On the new PC install:

Node.js (LTS)

Git (optional, only if you clone from GitHub)

To check Node is installed:

node -v
npm -v


2) Install dependencies

In the client folder run:


npm install


This recreates node_modules from package.json.

3) Start the dev server

npm run dev


4) Start the dev server

npm run dev

Local: http://localhost:5173


Install Tailwind packages

5) npm install -D tailwindcss @tailwindcss/cli @tailwindcss/postcss


6) Check project files

postcss.config.js

export default {
  plugins: {
    "@tailwindcss/postcss": {},
  },
};

 7) or 
 
cd CityLink/client
npm install
npm install -D tailwindcss @tailwindcss/postcss @tailwindcss/cli
npm run dev


8) Install Xml reader : 

npm install fast-xml-parser
